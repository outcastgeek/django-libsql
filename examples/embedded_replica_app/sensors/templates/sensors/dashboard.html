üöÄ Run Simulations

Execute these management commands to test different scenarios:

# Single-threaded simulation (30 seconds)
python manage.py simulate_sensors --duration 30

# Multi-threaded simulation (4 threads, 30 seconds)
python manage.py simulate_sensors --threads 4 --duration 30

# High-frequency mode (10 sensors, 8 threads)
python manage.py simulate_sensors --sensors 10 --threads 8

# Test ALL modes automatically
python manage.py test_all_modes

# With no-GIL Python (true parallelism)
PYTHON_GIL=0 python -X gil=0 manage.py simulate_sensors --threads 8

  • Aggregated data updates automatically every few seconds
