üîß Command Line Benchmarks

For more control, run benchmarks from the command line:

# Run basic benchmark (CRUD test with default settings)
python manage.py run_benchmark

# Run with specific parameters
python manage.py run_benchmark --test crud --operations 1000 --threads 4

# Run ALL modes automatically (no manual intervention!)
python manage.py benchmark_all_modes

# With no-GIL Python (free-threading)
PYTHON_GIL=0 python -X gil=0 manage.py run_benchmark --threads 8

# With embedded replica mode
USE_EMBEDDED_REPLICA=1 python manage.py run_benchmark

# Compare GIL vs No-GIL performance
python manage.py compare_gil_performance
üîç What to observe:
  • GIL vs No-GIL: No-GIL shows better scaling with multiple threads
  • Thread Scaling: Performance should improve with more threads (especially without GIL)
  • Embedded vs Remote: Embedded replica typically faster for reads
  • Operation Types: Different patterns show different performance characteristics
